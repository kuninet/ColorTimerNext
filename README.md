# ColorTimer Next

ColorTimer の次期バージョンプロジェクトです。
IoTデバイス (ESP32) のボタンを押すことで、作業の開始・終了をクラウド（AWS）へ送信し、Webアプリでその履歴や統計情報を確認することができます。

## プロジェクトの目的と要件

本プロジェクトは以下の要件・前提のもとで開発を進めています。
詳細な機能要件、非機能要件については [要件.md](./docs/要件.md) を参照してください。

* **シンプルな実装**: テスト的に1台の機器からAWSにデータを送ることを想定しており、エッジ側もクラウド側も限りなくシンプルな設計・実装を行います。
* **限りなく無料枠**: 個人利用のためのプロジェクトであり、AWSの利用についてはAWS無料枠（Always Free / 12ヶ月無料枠）に全て収めることを目標としています。
* **オープンソース**: 本プロジェクトはオープンソース（OSS）として公開・管理します。そのため、シークレットや認証情報はコードに直書きせず、安全に管理する仕組みを取り入れています。
* **AIエージェントの活用**: 機能開発においてAIエージェントチームを活用しており、裏付け調査などのタスクには軽量なモデル（Gemini Flash等）を使用するルールで運用しています。

## リポジトリ構成 (予定)

* `/edge` : ESP32用プロジェクト（PlatformIOなどを用いたエッジ側のコード）
* `/backend` : AWSリソース構築用（AWS SAM / CDKなど構成）
* `/frontend` : Webアプリケーション（SPA等）
